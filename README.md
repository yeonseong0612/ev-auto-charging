# ⚡ EV-Auto-Charging
> 전기차 충전구 인식 및 자동 충전을 위한 End-to-End 솔루션 프로젝트
> End-to-end solution project for electric vehicle charging port recognition and automatic charging


## 프로젝트 개요(Project Overview)
> 본 프로젝트는 전기차 충전 인프라의 편의성과 접근성을 높이기 위해, 비전 기반 충전구 인식 및 로봇팔 제어를 통해 자동으로 충전 커넥터를 연결하는 시스템을 개발합니다.  
> This project aims to develop an automated EV charging system by detecting charging ports through computer vision and controlling a robotic arm.

## 주요 기능(Main features)
- YOLO 기반 충전구 탐지 및 세그멘테이션  
  (YOLO-based charging port detection and segmentation)
- 중심점 및 거리 추정  
  (Center point & depth estimation)
- 로봇팔 제어 및 충전 커넥터 삽입  
  (Robotic arm control and coupler insertion)
- 앱 & 시뮬레이터와 통합 연동  
  (Integration with mobile app and simulator)
